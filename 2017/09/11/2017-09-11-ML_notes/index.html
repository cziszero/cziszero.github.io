正则项

由于模型复杂度高时容易导致过拟合，所以要加入一个正则项。
正则化项一般是模型复杂度的单调递增函数，模型越复杂，正则项应越高。

ROC和AUC

ROC曲线:
x轴为假阳性: $FPR = \frac{NP}{N}$
y轴为真阳性: $TPR = \frac{TP}{P}$
通过调整截断点产生, 零点对应最高的得分.
ROC比P-R曲线的优点在于当正负样本的分布发生变化时, ROC曲线的形状能基本保持不变, 而P-R曲线一般会发生剧烈变化.

优化算法

梯度下降

ref

  1. GD: $\theta = \theta - \lambda d\theta$
  2. SGD
  3. mini-batch SGD
  4. Momentum:
    $v_t = \gamma v_{t-1} + \lambda d\theta$
    $\theta = \theta - \lambda v_t$
  5. AdaGrad:
    增大更新频率低的参数的学习率, 使用历史梯度的平方和来衡量不同参数梯度的稀疏性, 乘在学习率上.
    $\theta = \theta -\lambda\frac{1}{\sqrt{\sum_{t=1}^T(d\theta_t)^2+\epsilon}}d\theta$
  6. RMSprop:
    将AdaGrad中时历史梯度平方和换为指数衰减平均
  7. Adam:
    AdaGrad + RMSprop
    使用指数衰减平均引入一阶矩和二阶矩

正则化

从Bayes的角度

  • L1正则化相当于引入拉普拉斯先验, 即参数w服从拉普拉斯分布
  • L2正则化相当于引入高斯分布, 即参数w服从高斯分布
    可以从最大后验似然函数推导出来
    ref

    其他角度

    L2 起到 weight decay的作用， 噪声的梯度的方向一般是不固定的， 所以会无法抵消掉weight decay的作用， 而非噪声的参数的梯度一般要朝一个固定的方向进行， 所以可以抵消weight decay的影响。

逻辑回归

公式推导

  1. 二分类 ref：
    forward
    $Loss = -\Sigma_i(ylog(p)+(1-y)log(1-p))$
    $p = \sigma(z) = \frac{1}{1-e^{-z}}$
    $z = wx$
    backward
    $dp = -\frac{y}{p} + (1-y)\frac{1}{1-p}$
    $dz = p-y; p = \sigma(x,w)$
  2. softmax ref:
    下面写成单样例的形式， 对于整个batch相应的加和即可
    forward:
    $Loss = -\Sigma_iy_{i}\log(p_i)$
    $p_i = softmax(z_i) = \frac{e^{z_i}}{\Sigma_j{e^{z_j}}}$
    $z = wx$
    backward:
    由于对于每个样例只有一个正确答案，即只有一个$y_i$为1, 其他全为0， 在Loss中不体现， 所以设该样例的真实类别为i， 则有$Loss = -\log{(p_i)}$
    $dp_i = -\frac{1}{-p_i}$
    $dz_i = p_i - y_i$ , $y$使用one-hot编码

树模型

分裂指标

  1. ID3 最大信息增益
    数据集D的经验熵： $H(D) = -\Sigma_{k=1}^{K}{\frac{|C_i|}{|D|}\log{\frac{|C_i|}{|D|}}}$
    根据特征A分裂后得到条件经验熵： $H(D|A) = \Sigma_i{\frac{|D_i|}{|D|}H(D_i)}$
    信息增益： $g(D,A) = H(D|A) - D(D)$
    缺点: 偏好特征值多的特征，使得泛化性能不好
    特点: 使用信息增益作为特征选择依据, 每个特征只用一次, 包含预剪枝(如果某次所有特征的信息增益都小于某个阈值, 则停止生成).
  2. C4.5 最大信息增益比
    改进最大信息增益的问题，添加特征值的惩罚
    $g_r(D,A) = \frac{g(D,A)}{H_A(D)}$
    其中$H_A(D)$表示按A特征划分的熵
  3. CART：
    分类问题中根据gini指数描述数据的纯度，$Gini(D) = 1-\Sigma_{k=1}^{K}{(\frac{|C_k|}{|D|})^2}$
    CART是二叉树， 在特征A的条件下, 集合D被分成两类, 其基尼指数定义为: $Gini(D,A)=\frac{|D_1|}{|D|}Gini(D_1)+\frac{|D_2|}{|D|}Gini(D_2)$. 选择使 $Gini(D,A)$ 最小的特征值进行分类.
