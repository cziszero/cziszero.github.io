树的前序、中序、后序遍历

  1. 遍历 洛谷1030 求先序排列： 给出一棵二叉树的中序与后序排列。求出它的先序排列。
  2. Leetcode 145 Binary Tree Postorder Traversal 非递归求二叉树的后序遍历. 如果先序遍历时先访问右儿子再访问左儿子, 则先序遍历和正常的后序遍历结果一样, 有几种实现非常优雅, 非递归实现时重点关注遇到NULL该怎么办, 思路就比较清晰了, 因为如果不是NULL, 那肯定是访问左儿子.
  3. Leetcode 232 Implement Queue using Stacks 非递归实现很容易, 可以练习下非递归实现

二叉树

  1. Leetcode 687.Longest Univalue Path 很简单的一道题, 但需要注意的是过程量和结果要分别维护
  2. Leetcode 226.Invert Binary Tree 简单题, 交换左右儿子
  3. Leetcode 617.Merge Two Binary Trees 简单题, 遍历下就好了
  4. Leetcode 669.Trim a Binary Search Tree 熟悉BST的结构
  5. Leetcode 971.Flip Binary Tree To Match Preorder Traversal
  6. Leetcode 114.Flatten Binary Tree to Linked List 二叉树转为链表, 有非递归的简单解法

trie

模板

题目

  1. Leetcode 211.Add and Search Word
  2. leetcode 720.Longest Word in Dictionary 先排序然后检索前缀是否在里面, 用HashMap做检索的话时间复杂度是$O(nklogn)+O(nk)$, 其中n是单词表中单词的个数, k是单词的平均长度. 然而和前缀有关, 所以可以用Trie做, 插入时间复杂度$O(nk)$, 检查时间复杂度是$O(nk)$
  3. leetcode 208.Implement Trie (Prefix Tree)
  4. leetcode 212.Word Search II 因为有前缀, 所以可以用Tire优化
